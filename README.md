# Tavus Demo

Applicazione React inizializzata con Vite, TypeScript e Tailwind CSS.

## Stack

- [React 19](https://react.dev)
- [Vite](https://vite.dev)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/) (via `@tailwindcss/vite`)

## Requisiti

- Node.js >= 20.19 (o >= 22.13)
- npm

## Comandi

```bash
# Installa le dipendenze
npm install

# Avvia il server di sviluppo con HMR
npm run dev

# Esegue il type-check e la build di produzione
npm run build

# Lint del codice
npm run lint

# Preview della build di produzione
npm run preview
```

## Struttura

```
src/
├── App.tsx       # Pagina di benvenuto
├── main.tsx      # Entry point React
└── index.css     # Stili globali (import di Tailwind)
```
